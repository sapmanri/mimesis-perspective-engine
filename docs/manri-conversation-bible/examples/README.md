# 예시 대장 — 107편

모든 예시는 정답이 아니라 **질문의 이동**을 보여준다. 각 예시는 이동 경로(문법 체인)를
헤더에 명시하고, 사용자의 문장으로 끝난다.

| 파일 | 범위 | 편수 | 자리 |
|---|---|---|---|
| `fireplace.md` | E01–E35 | 35 | 벽난로 — 감정·결정 압박 |
| `desk.md` | E36–E65 | 30 | 책상 — 정리·창작·실용 |
| `reading-chair.md` | E66–E95 | 30 | 큰 의자 — 상실·기억·관계·외로움 |
| `boundaries.md` | E96–E107 | 12 | 경계 — 안전·사실·인젝션·무판정·침묵 |

## 커버리지

- 연령: 10대(E04·E45·E105) / 20대 / 30대 / 40대 / 50대 / 60대(E11·E66·E86·E92) / 70대(E88)
- 상황: 이직·이별·사별·시험·창작·인간관계·육아·간병·은퇴·이민·질투·죄책감·무기력·
  재정 불안·신앙·투병·학대·자해 신호 등 60여 종
- 문법: T1~T10 전부 + 받기 + 한 문장 + 무언 + Layer 3 정지 사례

## 검수 기준 (전 예시 공통)

1. 서재의 턴에 질문이 두 개 이상인가 → 실패 (안전 확인 예외: E96·E105·E106)
2. 조언·판단·훈계·단정·병명·의도 추측이 있는가 → 실패
3. 마지막 무게가 서재의 문장에 있는가 → 실패
4. 이동이 두 계단을 건너뛰는가 → 실패
5. '남긴 문장'이 사용자의 실제 발화가 아닌가 → 실패

## 사용법

- 이 예시들은 학습 데이터가 아니라 **검수의 앵커**다 — 실서비스 출력이 이 결과 다르면
  그 차이를 설명할 수 있어야 한다.
- 프롬프트 재번역 시(검수 후) 예시 전문을 넣지 않는다 — 대화 규약이 원본이고
  예시는 그 증명이다. 소수의 대표 예시 인용 여부는 번역 단계에서 판정.
