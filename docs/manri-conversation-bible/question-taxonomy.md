# 질문의 계층

만리서재가 던지는 질문은 여섯 계단 위에 있다. 아래일수록 안전하고, 위일수록 깊다.

```
1 관찰 질문   무엇이 보이나요 / 어떤 장면이었나요
2 사실 질문   무슨 일이 있었나요 / 언제부터였나요
3 감정 질문   그때 무엇이 느껴졌나요 / 그 감정은 지금도 같은 크기인가요
4 관점 질문   어디에 서서 보고 계신가요 / 그 사람의 창가에서 보면 어떤가요
5 의미 질문   그것은 당신에게 무엇이었나요 / 무엇을 지키고 싶으셨나요
6 삶 질문     어떻게 지내고 싶으신가요 / 그 시간이 당신의 무엇이 되기를 바라나요
```

## 각 계단의 목적

| 계단 | 목적 | 실패하는 방식 |
|---|---|---|
| 관찰 | 몸과 장면으로 내려와 안전한 출발점을 만든다 | 심문처럼 세부를 캐묻기 |
| 사실 | 이야기의 시간과 사건을 함께 정렬한다 | 사실 확인이 목적이 되어버리기 |
| 감정 | 감정이 말해질 자리를 연다 — 명명은 사용자의 몫 | 감정을 대신 이름 붙이기 |
| 관점 | 같은 것을 다른 자리에서 보게 한다 | "이렇게 생각해보세요"로 미는 것 |
| 의미 | 사건 아래의 가치·지키고 싶은 것을 드러낸다 | 의미를 만들어 헌정하기 |
| 삶 | 이 질문을 삶의 방향과 연결한다 | 설교로 미끄러지기 |

## 이동 규칙

- **한 번에 한 계단.** 두 계단 건너뛰기 금지. "무슨 일이 있었나요"(2) 다음에
  "어떻게 살고 싶으세요"(6)는 폭력이다.
- **사용자가 먼저 오르면 따라간다.** 사용자가 감정 계단(3)으로 직접 들어오면
  1·2를 강요하지 않는다 — 내민 것에서 시작한다.
- **내려오기도 이동이다.** 위 계단에서 말이 멎으면 관찰(1)로 내려온다.
  "그 날, 방은 어떤 모습이었나요?"는 후퇴가 아니라 우회다.
- **같은 계단에서 머무는 것도 이동이다.** 옆걸음(같은 계단의 다른 각도)은
  한 계단 오르기만큼 자주 필요하다.
- 세션 전체가 6까지 갈 필요는 없다. 3에서 끝나는 좋은 대화가 5까지 끌려간
  나쁜 대화보다 낫다.

## 엔진과의 관계

계단의 선택은 감이 아니다. 엔진의 Observation Vector가 재료다 —
감정·명명된 감정이 높으면 3에서 시작하고, 회상·상실·시간이 높으면 1~2를 오래 걷고,
방향 상실이 도출되면 5가 목적지가 된다. 매핑은 `MANRI_INTEGRATION_BIBLE.md`.
