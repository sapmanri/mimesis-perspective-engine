# 재질·물성·색채 기준

| 항목 | 사양 | 낡음의 정도 |
|---|---|---|
| 벽 | 석회 플라스터, 손흙손 자국 미세하게 | 모서리와 스위치 주변만 손때로 어둡다 |
| 바닥 | 오크 장부맞춤 판재, 폭 14cm | 러그 밖 통행선만 광이 죽어 있다 |

| 책장 | 호두나무, 고정 선반, 동벽에 사다리 레일(놋쇠) | 선반 앞모서리만 둥글게 닳음 |
| 흔들의자 천 | 리넨 방석, 황토색 | 앉는 자리 중심이 눌려 색이 반 톤 옅다 |
| 큰 의자 천 | 양모 벨벳, 이끼색 | 팔걸이 끝 보풀, 오른쪽만 더 닳음 |
| 담요 | 오트밀색 양모, 무지(체크 금지) | 접힌 자국이 남아 있는 정도 |

| 타자기 | 강판+법랑, 무광 검정 | 자주 쓰는 키(ㅇ, ㅅ 위치)만 반질하게 닳음 |
| 금속 | 놋쇠(스탠드·레일·문손잡이), 흑철(부지깽이·경첩) | 놋쇠는 만지는 부분만 밝다 |
| 유리 | 창=구식 판유리(미세한 물결 왜곡), 스탠드 갓=초록 유리 | 완벽히 투명하지 않다 |
| 벽난로 | 회색 사암 서라운드, 내부 내화벽돌 | 개구 상단에 그을음 번짐 고정 |
| 먼지 | 빛줄기 안에서만 보인다. 표면에 쌓인 먼지는 없다 | — |

**물성 총칙**: 새것처럼 보이면 안 된다. 빈티지 소품 전시장처럼 과장해도 안 된다.
누군가 오래 사용해온 실제 공간 — 낡음은 "사용된 곳에만, 사용된 만큼".

## 5. 시간·날씨의 창밖 색

**창밖 풍경은 의도적으로 비확정이다** (Vase 판정 2026-07-20):

```text
Exterior view is intentionally undefined.
Nothing outside the windows in this image becomes canonical.
```

아래 표는 유리에 맺히는 **색과 밝기의 지침일 뿐**, 창밖에 무엇이 있는지(산·호수·숲·
거리)는 어떤 문서도 정하지 않았다. 촬영된 컷의 창밖에 우연히 그려진 풍경은 캐논이
아니며, image reference로도 상속하지 않는다. **창밖 대장은 첫 낮 컷 전에 판정한다.**

| 상태 | 창밖 | 실내 보정 |
|---|---|---|
| 새벽 | `#6E7B8B` | 실내 전체 -1.5스톱, 벽난로 잔불만 |
| 오전 | `#D8E2E6` | 기준 노출 |
| 오후 | `#DCD9C8` | 북창광이 반 톤 따뜻해짐 |
| 저녁 | `#8B6B4F` | 불·스탠드가 주광원으로 교대 |
| 밤 | `#1C2430` | 광원 웅덩이 3개만 남음 |
| 비 | `#39465A` | 실내 -0.5스톱, 불·스탠드 상대적 강조 |
| 눈 | `#AEB8C2` | 창가만 +0.5스톱, 차고 고요한 반사광 |

## 6. UI와 공간의 충돌 방지 규칙

- UI 종이 `#FAF8F4`와 글자 `#2B2925`는 공간 팔레트의 일부다 — UI만의 색을 새로 만들지 않는다.
- 공간의 강조색(주황·초록·청색)을 UI 요소(버튼·링크)에 쓰지 않는다. UI 강조는 굵기와 여백으로.
- 어두운 장면 위 UI는 색을 바꾸지 않고 그림자(shadow)로만 분리한다.
