# 조명 설계와 카메라 문법

## 조명 — 공간 전체

- 광원은 넷뿐이다: **북창(확산 자연광) · 동창(약한 자연광) · 벽난로 · 스탠드 2기**
  (B석 책상 스탠드, C석 플로어 스탠드) + 저녁 한정 맨틀 초 1자루. 천장 조명은 존재하지 않는다.
- 자연광 방향: 북창은 하루 종일 직사 없는 확산광(이 공간의 기준광). 동창은 오전에만 낮은 사광.
- 공간의 정체성 = **세 개의 빛 웅덩이**(불 1900K / 북창 5000K / 스탠드 2400K)가
  어두운 중간지대를 사이에 두고 공존하는 것. 웅덩이 사이를 밝히지 않는다.

### 시간대별

| 시간 | 지배 광원 | 규칙 |
|---|---|---|
| 새벽 | 벽난로 잔불 | 창 `#6E7B8B`, 실내 형태만 겨우 읽힘 |
| 오전 | 북창 | 동창에서 낮은 사광 1줄 (바닥에 창틀 그림자) |
| 오후 | 북창 | 광 반 톤 따뜻, 그림자 약간 길어짐 |
| 저녁 | 불 + 스탠드 | 자연광과 인공광의 교대 시간, 초 점화 |
| 밤 | 불 + 스탠드 | 웅덩이 밖은 형태가 실루엣으로만 |

- 그림자 밀도: 중간지대의 그림자는 뭉개지지 않고 형태를 유지한다(완전 검정 금지, 최암부 `#1A1611`).
- 인물이 없어도 살아 보이는 조명: 웅덩이 가장자리의 미세한 흔들림(불), 종이 위 나뭇가지
  그림자의 극히 느린 이동(북창), 스탠드의 호흡(±3%). 이 셋이 공간의 맥박이다.
- 대화 중 화면 규칙: 밝기 변동 합계 ±4% 이내. 사용자가 글을 읽고 쓰는 동안
  빛이 사건을 만들지 않는다.

## 카메라 문법

### 기본

| 항목 | 값 |
|---|---|
| 렌즈 | 35mm (풀프레임 환산) 단일 — 전 장면 공통 |
| 화각 | 수평 약 54° |
| 높이 | 서서 150cm, 착석 A 100 / B 115 / C 105cm |
| 시점 | 1인칭에 가까운 3인칭. 사용자의 눈이거나 반 걸음 뒤 |
| 이동 속도 | 0.7m/s (느린 걸음), 가감속 0.8s ease |
| 흔들림 | 진폭 0.3° 이하, 주기 0.2Hz (호흡이지 손떨림이 아님) |
| 심도 | 기본 f/4. B석 f/4~5.6, C석만 f/2.8 허용 |
| 포커스 이동 | 1.2s 이상, 한 장면에 1회 이하 |

### 장면 규칙

- 좌석 접근: `spatial-layout.md` 5절의 동선·시간을 따른다. 접근 중 프레임에
  다른 좌석의 빛 웅덩이가 최소 1회 스친다 (같은 방의 증거).
- 착석 후 고정 구도: 각 좌석 문서의 값으로 고정. 대화 내내 카메라는 움직이지 않는다.
- 첫 방문: 문 → 기준 컷 3초 정지 → 접근. / 재방문: 기준 컷부터 시작(문 생략), 정지 1초.
  같은 공간이 "이미 아는 곳"이 되는 감각은 이 생략에서 온다.

### 금지

- 드론 문법 (부감 하강, 상승 리빌, 궤도 회전)
- 광고 문법 (빠른 달리, 속도 램프, 화려한 트랜지션)
- 게임 트레일러 문법 (과장 원근, 초광각, 카메라 플레어)
- 과다 보케 (배경이 원반으로 뭉개지는 수준), 시네마틱 슬로모션
- 장면마다 공간 구조가 달라지는 생성 (layout lock 위반 — 폐기 후 재생성)
