# 움직임 · 사운드 · 시간과 계절

## 1. 움직임의 규칙

원칙: 공간은 살아 있되 산만하지 않다. **사용자가 생각 중일 때 공간이 감정을 대신 연기하지 않는다.**

| 움직임 | 허용 조건 | 속도·빈도 |
|---|---|---|
| 불꽃 | 상시 (A석 광원) | 진폭 작게, 밝기 영향 ±4% 이내 |
| 커튼 | 바람 날씨에만 | 6~10s 주기 미동, 펄럭임 금지 |
| 먼지 | 빛줄기 안에서만 | 10초에 1~2개 입자가 느리게 통과 |
| 빗방울 | 비 날씨, 동창 유리 | 흘러내림 느리게, 화면당 동시 3~5줄 |
| 눈 | 눈 날씨, 창밖 | 수직 낙하 느리게, 창밖에서만 |
| 페이지 | 사용자 행위에 의해서만 | 스스로 넘어가지 않는다 |
| 증기 | 머그에 뜨거운 것이 있을 때만 | 가늘고 곧게, 1분 내 소멸 |
| 나뭇가지 그림자 | 낮, 북창 종이 위 | 10분에 눈치챌 정도의 이동 |
| 흔들의자 | 착석 직후에만 | 3회 진동 후 정지 |
| 스탠드 불빛 | 상시 | ±3% 호흡, 주기 6~8s |

동시 활성 움직임은 한 프레임에 3종 이하. 사건성 움직임(무언가 떨어짐, 문이 열림 등)은 없다.

## 2. 사운드 디자인

원칙: **음악은 기본값이 아니다. 침묵이 만리서재의 중요한 일부다.** 전체 음량은 낮고(배경 -38LUFS
기준), 언제든 완전히 끌 수 있다.

### 공통 (방 전체)

- 나무의 작은 수축음: 1~2회/분, 위치 무작위
- 벽시계: 진자 소리는 **밤에만** 들린다 (낮에는 시각적 존재만)
- 바람: 바람 날씨에만, 처마를 지나는 낮은 소리

### 좌석별

| 좌석 | 가까운 소리 | 먼 소리 |
|---|---|---|
| A 벽난로 | 장작 크래클, 착석 시 의자 삐걱 3회 | 방 건너 먼 비(비 날씨) |
| B 책상 | 종이 스침(입력 시), 연필 구름(드묾) | 남벽 시계(밤), 먼 불소리 |
| C 큰 의자 | 동창의 비, 담요 스침, 페이지(드묾) | 먼 천둥(10분 1회 이하) |

- 타자기 소리·컵 내려놓는 소리: 사용자 행위에 대응할 때만. 장식음 금지.
- 소리는 좌석 간 이동 시 크로스페이드 2s.

## 3. 시간 · 날씨 · 계절

원칙: **가구와 구조는 절대 바뀌지 않는다.** 변하는 것은 빛, 창밖, 사물의 작은 배치,
불의 세기, 공기의 밀도뿐이다.

| 상태 | 변하는 것 (허용 전량) |
|---|---|
| 새벽 | 창 `#6E7B8B`, 잔불, 실내 -1.5스톱, 소리 거의 없음 |
| 오전 | 동창 사광 1줄, 북창 기준광, 불은 낮게 |
| 오후 | 광 반 톤 따뜻, 그림자 길어짐 |
| 저녁 | 초 점화, 불·스탠드로 교대, 창 `#8B6B4F` |
| 밤 | 웅덩이 3개만, 시계 소리 등장, 창 `#1C2430` |
| 맑음 | 기준 |
| 비 | 창 `#39465A`, 동창 빗줄기, 실내 -0.5스톱, 먼 비 소리 |
| 눈 | 창 `#AEB8C2`, 창밖 수직 낙설, 창가 +0.5스톱, 소리 더 조용 |
| 겨울 | 불이 항상 타고 있음, 담요 사용감, 창틀 아래 미세 결로 |
| 봄 | 창가 화분 잎이 조금 자람, 커튼 미동 빈도 소폭 증가, 불은 잔불 |

- 사물의 "작은 배치" 허용 범위: 머그의 위치(테이블 위 10cm 반경), 푸른 책의 펼침/덮임,
  담요의 개킴/펼침, 종이 매수. 그 외 오브젝트 이동 금지.
