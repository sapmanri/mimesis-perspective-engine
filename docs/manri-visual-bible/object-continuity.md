# 오브젝트 연속성 대장

프롬프트 continuity lock의 원천이다. 위치 좌표는 `spatial-layout.md` 좌표계.

## 규칙

- 이 12종은 **삭제·이동·교체 금지**. 유일한 예외는 `motion-and-sound.md` 3절의
  "작은 배치" 허용 범위(머그 10cm, 책 펼침/덮임, 담요 개킴, 종이 매수).
- 새 오브젝트 추가는 게놈의 형질 추가와 같은 절차 — 설계자 판정으로만, 이 대장에 등록 후 사용.
- 생성 결과 검수 시 이 표가 체크리스트다: 12종 중 프레임에 들어와야 할 것이 다르게 나오면 폐기.
