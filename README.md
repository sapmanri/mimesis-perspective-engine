# MIMESIS Perspective Engine

삽만리 Observation Genome을 독립 검증하는 질문형 답변 엔진. 2D Walk(`mimesis-sight-path`)와 별개의 데모 프로젝트다.

> 정확한 답을 먼저 만든다.
> 그 답을 삽만리의 시선으로 다시 바라본다.
> 그러나 질문의 본래 목적은 흐리지 않는다.

## 문서

- [설계 정본 V1.1](docs/SAPMANRI_PERSPECTIVE_ENGINE_V1.md) — 게놈 스키마, 질문 유형별 처리, 다양성 규칙, 안전·사실성 계약, 프롬프트 구조, MVP 명세, 테스트 질문 20개와 합격 기준

## 상태

설계 확정 단계. 구현(프롬프트·API·UI)은 아직 시작하지 않았다.
